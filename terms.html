2. Description of Service

HorizantalTabs is a Chrome browser extension that provides horizontal tab management. It displays open tabs in a scrollable strip, allowing users to search, switch, reorder, and organize tabs. A paid "Pro" tier offers additional features including tab thumbnails, session saving, undo close, and multi-window support.
